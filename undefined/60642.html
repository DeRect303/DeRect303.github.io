另一点是’r‘被’w’给覆盖了，原因和上面有关，（因为这点得修改脚本了），因为0索引没被确认，但确认了0索引的值，就会覆盖掉前一个索引指向的值（因为赋值语句放在了检索值处）

EXP：

import idaapi

import idautils

import idc

import ida_dbg

import ida_funcs

start=0x00001155

end=0x00001479

dic={}

for ea in range(start,end):

point=idc.get_wide_dword(ea)

if point==0xC08348F8:

​ i=ea+4

​ index=idc.get_wide_byte(i)

if point==0x3C00B60F:

​ i=ea+4

​ value=idc.get_wide_byte(i)

​ if index!=None:

​ dic[index]=value

​ else:

​ dic[0]=value

​ index= None

​ value= None

print(dic)
